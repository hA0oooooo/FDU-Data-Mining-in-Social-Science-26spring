# 主课补习对学生成绩、认知与情绪的影响

本项目使用中国教育追踪调查（CEPS）2013-2015 年七年级追踪数据，研究主课补习对学生学业成绩、一般认知能力和短期负面情绪的影响。研究关注的核心问题是：补习带来的收益是否主要体现为考试科目的成绩提升，还是也能反映在更一般的认知能力上；同时，补习是否伴随学生短期情绪压力的变化。

项目区分任意课外补习、任意主课补习和语文、数学、英语三类单科补习。分科成绩模型控制其他主课补习，并纳入学习基础、学生个体特征、家庭教育背景和学校环境等协变量，以减少补习组合和家庭投入差异带来的混杂。实证方法包括 DID、IPW-DID 和 DR-DID，并通过样本范围、模型设定和跨学科检验考察结果稳健性。

当前结果显示，数学补习对数学成绩的提升最稳定，英语补习存在正向收益信号，语文补习没有独立显著效应；任意主课补习对总成绩有正向影响，但没有证据表明其显著提高一般认知能力。短期负面情绪方面，平均负面情绪呈微弱正向信号，高负面情绪结果不显著。整体上，主课补习的收益更集中于面向考试的学科训练，补习对学生成长的影响需要从成绩、认知和心理状态三个层次理解。

项目目录按照正式研究流程组织：`scripts/` 保存数据处理、模型估计、稳健性检验和图表生成代码，`outputs/` 保存主要估计结果和样本统计表，`figures/` 保存论文图表。
